## Purpose
Visual replay of all system activity. Immutable, sponsor-grade, and publicly viewable.

## Layers
- Replay: shows task flow, custody, rotation, and signals
- Grades: shows entity grades per sector
- Ads: shows sponsor-grade ads from green-grade entities
- Treasury: shows fines, fees, and ledger entries

## Inputs
- Logs from `/99_Logs/`
- Grades from `/manual/graded_system.md`
- Sector map from `city_grid.json`

## Outputs
- Screenshots and video replay
- QR-coded grade display
- Public vote interface
- Ad placement dashboard

## Mandate
- Must run 24/7
- Must show provenance of every report
- Must be sponsor-grade and immutable
